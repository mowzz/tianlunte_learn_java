Java与操作系统

Basic Structure

  1. Softwares can be separated into several layers:

    Kernel
    Operating system that is always running
    System programs
    Associated with operating system, but not necessarily part of it
    MiddleWare
    A set of software frameworks providing additional services to Application developers
    Application programs
    Not associated with operating system

  2. Computer System Structure

    A modern general-purpose computer typically consists of one or more CPUs and a number of device controllers connected through a common bus which also provides access to shared memory

    CPUs and device controllers can execute in parallel and compete for memory access

    A memory controller synchronizes access to memory, ensuring orderly access
